## 一、排盘算法（四柱推导）

| 柱 | 依据 | 算法 |
|---|---|---|
| **年柱** | 《万年历》，以立春为年界（非春节） | `(年份-4) % 10` 得天干，`(年份-4) % 12` 得地支 |
| **月柱** | 五虎遁月法（"甲己之年丙作首"） | 年干→推月干起点，节气定月支 |
| **日柱** | 基准日推算法 | 以2000-01-01（庚辰日）为锚点，算天数差取60甲子余数 |
| **时柱** | 五鼠遁时法（"甲己还加甲"） | 日干→推时干起点，时辰定地支 |

**经典出处：** 年柱/月柱/时柱的推导规则来自《渊海子平》《三命通汇》等命理典籍。

## 二、五行分析

| 概念 | 依据 |
|---|---|
| **天干五行** | 甲乙属木、丙丁属火、戊己属土、庚辛属金、壬癸属水（《五行大义》） |
| **地支五行** | 子亥属水、寅卯属木、巳午属火、申酉属金、辰戌丑未属土 |
| **地支藏干** | 每个地支暗藏1-3个天干（如子藏癸、丑藏己癸辛），来自《滴天髓》 |

## 三、十神体系

这是最核心的分析维度。原理：

```
日主（日柱天干）与其他天干的五行生克关系 → 十种角色
```

| 关系 | 同性 | 异性 |
|---|---|---|
| 同我（比和） | 比肩 | 劫财 |
| 我生（泄） | 食神 | 伤官 |
| 我克（耗） | 偏财 | 正财 |
| 克我（制） | 七杀 | 正官 |
| 生我（扶） | 偏印 | 正印 |

**经典出处：** 《子平真诠》——十神论命的根本大法。

## 四、日主强弱判定

页面用的是简化版：

```
日主同类五行 + 生我（印星）五行的总得分 ≥ 4 → 偏强
                                    ≤ 2 → 偏弱
                                    else → 中和
```

**完整版**还需要考虑月令（得令/失令）、通根（地支藏干）、透干等因素，这里做了简化处理。
